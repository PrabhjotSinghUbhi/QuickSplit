Color System Overview (Tailwind + HEX + Roles)

1. Primary Accent — Teal/Cyan tones

Purpose: Core branding and interaction color.

Usage: Logo, CTA buttons, highlights.

Classes:

from-teal-500 → #14b8a6 (gradient start)

to-cyan-600 → #0891b2 (gradient end)

text-cyan-400 → #22d3ee (links, logo accents)

border-cyan-500 → #06b6d4 (borders, avatar rings)

2. Base Theme — Deep Charcoal/Navy (Dark Mode)

Purpose: Foundation for a modern dark aesthetic.

Usage: Layout background and text hierarchy.

Classes:

bg-gray-900 → #111827 (main background)

bg-gray-800/80 → #1f2937 (cards, overlays)

text-white → #ffffff (primary text)

text-gray-400 → #9ca3af (secondary text)

3. Functional / Status — Red & Green indicators

Purpose: Financial and status feedback (success/error).

Usage: Toasts, balance indicators, logout/error states.

Classes:

bg-red-600 / text-red-400 → #dc2626 / #f87171 (negative/debit/error)

bg-teal-600 / text-green-400 → #0d9488 / #4ade80 (positive/credit/success)
